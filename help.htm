Subtitler 1.0
Help File

This is only a very short reference to the many function of Subtitler. More information will be added soon.

1. Keyboard shortcuts:

Keypad +  Inserts a new line at the current position
Keypad * or F2 Sets the player position at the starting frame of the current subtitle
Keypad - Opens the Edit subtitle dialog
Ctrl + Home Goes to the first subtitle
Ctrl + End Goes to the last subtitle
Space Starts / pauses the movie
F5 Marks in the starting frame
F6 Marks out the ending frame
Shift + (1-7) Sets a marker (1-7) at the player's current position
Ctrl + (1-7) Sets the player's current position at the marker (1-7)

2. Measuring the movie's frame rate (FPS)

If you have to change subtitles frame rate, you need to know your movie's frame rate first. Load a movie into the player and press the [Measure] button. Wait for about 20 seconds (by default) for the FPS to be measured. This feature is not very precise yet. If you think that the FPS is not measured correctly, try to increase the measuring interval from the Tools -> Options dialog, or set it manually.

3. Changing the frame rate of the subtitle file

If your subtitles are completely out of sync, the subtitle file might be using a frame rate, different from the FPS of the movie. Measure or set manually the movie's frame rate as described in point 2. Double-click on the either Start or End column of any subtitle to retrieve the Recalculate frames dialog. Select "Change the FPS of the entire movie" option and type or choose the new frame rate. Press OK to have all the frames recalculated.

4. Applying delays to the subtitles

If your subtitles appear earlier or later, you may need to apply certain delay in order to get them back in sync. Go to the first subtitle, from which on the movie is out of sync (usually the very first subtitle). Double-click on its Start frame value. Select the "Add seconds" radio box and type a delay in seconds (may be a negative number if the subtitles appear later). Ensure that the "Recalculate the frames for the rest of the movie" checkbox is checked and press OK.

5. Splitting subtitles

If you have a 2-CD movie, you may wish to split the subtitles file in two. Select the last title, which you wish to put in the first file, and select Edit -> Split subtitles... Type the names for the both files, press the [Save files] button and select a destination folder on your disk.

6. Merging subtitle files

When you have 2 subtitle files and you wish to merge them into one, use "Append subtitles" feature of Subtitler.  If you have the first file opened, select File -> Append subtitle file...  and point to the second file to be merged with the one, which is already opened.   The merging position will be marked with yellow. If you have the second file opened, you may wish to append it to the end of another file. Select File -> Append subtitles to... and point to the file, to which you wish to append the currently opened file. A suffix "-NEW" will be added to the name of the merged file.
